# hui-ui

> 我的Vue.js 2.0 网页UI组件。

注：本项目由个人开发，仅为了熟悉vue而开发，组件较简单，并不适合真实的网页开发。  

## 快速上手
样式使用了bootstrap，所以在页面中需要添加bootstrap.css  

### 运行&构建

``` bash
# 安装依赖
npm install

# 开发环境
npm run dev

# 生产环境
npm run build
```
